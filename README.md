# 🤖 Optimus AI SaaS v2.0

Многофункциональный AI-ассистент на базе **Gemini 2.0 Flash**, развернутый в Docker.

### 🧠 Ключевые возможности:
* **26 специализированных навыков:** От языков до системного администрирования.
* **Гибридная память:** PostgreSQL для чатов + ChromaDB для долгосрочного поиска (RAG).
* **Премиум-озвучка:** Интеграция с Microsoft Azure Cognitive Services.
* **Автоматизация (Cron):** Еженедельные саммари по английскому и напоминания по воскресеньям в 19:00 (Астана).
* **Монетизация:** Встроенная оплата через Telegram Stars (⭐️) после 50 сообщений.

### 🛠 Стек технологий:
* **Python 3.10** (aiogram/python-telegram-bot)
* **PostgreSQL** (хранение стейтов и истории)
* **ChromaDB** (векторная память)
* **Docker & Docker Compose**
* **Google GenAI & Azure Speech SDK**
